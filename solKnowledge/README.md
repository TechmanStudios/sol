# solKnowledge

This folder is the home for SOL working knowledge, consolidated knowledge, and proof packets.

## Subfolders
- `working/`: scratchpads, interim notes, active investigations, partial consolidations.
- `consolidated/`: curated, stable summaries and canonicalized knowledge artifacts.
- `proof_packets/`: audit-ready bundles (inputs, outputs, configs, run notes) supporting claims.

## Canonical references
- `consolidated/SOL_Agent_Architecture_Glossary_2026-02-14.md`: shared definitions for SOL agent/runtime workflows.

## Conventions (suggested)
- Prefer date-stamped files when iterating (e.g., `2026-01-24_topic_notes.md`).
- Keep proof packets self-contained and reproducible (include links to source data/artifacts).
